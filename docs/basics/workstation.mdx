title: Developer Workstation Setup Guide for Windows
---

## Overview
By the end of this guide, you'll have a Windows machine with the following tools installed:
- Ubuntu Linux
- VS Code

#### Prerequisites
- Windows 10 or higher

## Guide
### Step 1: Install Ubuntu Linux
<Steps>
  <Step title="Enable WSL">
    To enable WSL (Windows Subsystem for Linux), open the "Turn Windows features on or off" tool by searching for "features" in the Start menu, then scroll down and check the box next to "Windows Subsystem for Linux" and click "OK".
  </Step>
  <Step title="Install Linux with WSL">
    Install Linux with WSL by following this guide: [How to install Linux on Windows with WSL](https://learn.microsoft.com/en-us/windows/wsl/install)
  </Step>
</Steps>

### Step 2: Install VS Code
If you don't have an IDE, we highly recommend using VS Code. [Download VS Code](https://code.visualstudio.com/download) for Windows.
