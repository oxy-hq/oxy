Oxy ships as a single binary. Two commands cover every deployment scenario:

| | `oxy start` | `oxy serve` |
|---|---|---|
| **What it does** | Starts Oxy and manages PostgreSQL in Docker | Starts Oxy and connects to your PostgreSQL |
| **Database** | Automatic — Docker container, no setup | You provide `OXY_DATABASE_URL` |
| **Best for** | Local machine, quick VM deployment | Production with a managed database |
| **Requires** | Docker | External PostgreSQL |
